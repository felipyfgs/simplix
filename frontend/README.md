# Simplix — Frontend

Interface web do Simplix CRM, construída com Nuxt 3 e Nuxt UI.

## Setup

```bash
pnpm install
```

## Desenvolvimento

```bash
pnpm dev    # http://localhost:3000
```

## Produção

```bash
pnpm build
pnpm preview
```

> A API backend deve estar rodando em `http://localhost:8080`. Configure `NUXT_PUBLIC_API_BASE` no `.env` para apontar para outro endereço.
